Matthew Edgar

April 4, 2025

Introduction

The subject of my website is a smart AI assistant named Jarvis that I created as a personal project to explore the possibilities of browser-based artificial intelligence tools. Jarvis is inspired by the virtual assistant from the Iron Man franchise, but simplified and customized for real-world applications such as chatting, saving conversations, toggling between light and dark mode, and simulating intelligent conversation history.

Audience: Classmates, instructors, potential employers, and anyone interested in AI, web development, or tech tools.

Website URL:
https://matthewe-byui.github.io/wdd-130/personal-site-plan/jarvis.html

Goals / Objectives

  • Showcase skills in HTML, CSS, and JavaScript
  • Create a modular AI chat interface
  • Use localStorage for saving past chats
  • Implement light/dark mode
  • Ensure responsive and clean design

Design Process & Major Changes

  • Removed separate past conversations page and used sidebar interaction instead. It sometimes shows the past convos.
  • Refined chat layout to resemble a professional assistant
  • Improved light/dark mode compatibility
  • Enhanced transitions and layout consistency

Conclusion

This project improved my coding and debugging skills. It taught me the value of planning, testing, and iterating my design. These are valuable skills I’ll use in future software development and UI projects.
