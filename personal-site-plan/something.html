Introduction

The subject of my website is a smart AI assistant named Jarvis that I created as a personal project to explore the possibilities of browser-based artificial intelligence tools. Jarvis is inspired by the virtual assistant from the Iron Man franchise, but simplified and customized for real-world applications such as chatting, saving conversations, toggling between light and dark mode, and simulating intelligent conversation history.

The primary audience for my website includes classmates, instructors, potential employers, and anyone interested in AI, web development, or technology tools.

https://your-actual-site-link.com

Goals / Objectives

The goal of my website was to create a fully functional, visually appealing personal AI chat assistant that mimics basic features of a real virtual assistant. I wanted to:

  • Demonstrate skills in HTML, CSS, and JavaScript
  • Build a modular, extendable interface for AI interaction
  • Practice using localStorage to save and retrieve chat histories
  • Implement light/dark mode toggle
  • Create a responsive, clean design

Design Process & Major Changes

Originally, I envisioned a basic chatbot that would just display messages. However, the site evolved during development:

  • I removed a separate “Past Conversations” page and made past chats clickable from the sidebar.
  • I refined the design for the chat interface, making it resemble ChatGPT’s layout.
  • I added styling for light and dark modes and made sure both were visually clean.
  • I organized messages, improved load times, and added transitions.

These design choices made the experience more intuitive, clean, and functional.

Conclusion

I learned a lot through this project — not just about HTML and CSS, but also about how to use JavaScript effectively to create dynamic user experiences. I also improved my debugging skills and learned the importance of organizing my code and planning for future expansion.

I know these skills will help me in the future whether I’m working in software development, building tools for myself, or developing user interfaces in any environment.
